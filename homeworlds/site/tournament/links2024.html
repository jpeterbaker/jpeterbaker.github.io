  • Each group played a round-robin tournament with one game between each pair.
  • First player was selected randomly in each game.
  • Each win scored one point, each draw scored 1/2 point for both players.

Semifinal

BGA record
  • The 14 qualifying players were placed in two groups.

Final

  • The four finalists played a one-group round robin with two games between each pair.
  • Each player played first in one game against each opponent.
  • Each win scored one point, each draw scored 1/2 point for both players.
  • mstain became unavailable and did not finish some of their games.
